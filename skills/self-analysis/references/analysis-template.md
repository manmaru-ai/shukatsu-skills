## 分析時の指針

### 強みの発見

- 「当たり前にできること」が実は強みであることが多い
- 「みんなができる」と思い込んでいる特性に注目する
- 複数のエピソードに共通する行動パターンから強みを抽出する

### 具体的な言語化

抽象的な表現は避け、行動レベルで記述する。

| 避けるべき表現 | 具体化した表現の例 |
|----------------|-------------------|
| コミュニケーション力 | 相手の立場に立って情報を整理し、分かりやすく伝える力 |
| リーダーシップ | チーム全体の進捗を把握し、遅れている部分に自ら手を差し伸べる推進力 |
| 協調性 | 異なる意見を持つメンバー間の橋渡し役として合意形成を促す力 |
| 行動力 | 課題を見つけたら周囲の反応を待たずに自ら改善案を実行に移す力 |

### 弱みのポジティブ変換

| 本人の表現 | 就活で使える変換 |
|------------|-----------------|
| すぐ諦める | 状況を素早く判断し、効果的な方向転換ができる |
| 人に頼れない | 自走力が高く、主体的に問題を解決できる |
| 感情的になる | 物事に対して強い当事者意識を持てる |
| 行動が遅い | 多角的に検討してから確実に実行に移せる |
| 自分の意見がない | 多様な視点を尊重し、合意形成を促進できる |
| 心配性 | リスクを事前に察知し、予防策を講じられる |
| 完璧主義 | 高い品質基準を持ち、妥協せずに成果を追求できる |
| 飽き性 | 好奇心が旺盛で、新しい環境に素早く適応できる |

### 企業が評価する観点への翻訳

分析結果を以下の観点に紐づけて記述する。

- 主体性・当事者意識
- 課題発見・解決力
- チームワーク・協調性
- 論理的思考力
- 成長意欲・学習能力
- ストレス耐性・回復力
- 目標達成への粘り強さ
